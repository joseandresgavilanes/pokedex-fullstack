## 🚀 Getting Started

First, run the development server:

```bash
# Clone Repository
$ git clone https://github.com/joseandresgavilanes/pokedex-fullstack.git
# Go to server folder
$ cd pokedex-frontend
# Install Dependencies
$ npm i
# Run Aplication with ViteJS
$ npm run dev
```
